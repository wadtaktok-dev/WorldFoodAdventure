# WorldFood Adventure Website v2

Upload the contents of this folder to the root of the GitHub repository.

Files:
- index.html
- style.css
- script.js
- privacy-policy.html
- support.html
- .nojekyll
- robots.txt
- sitemap.xml
- assets/screenshots/

GitHub Pages:
Settings → Pages → Deploy from a branch → main → /(root)

Do not upload:
- google-services.json
- *.jks
- keystore.properties
- local.properties
- passwords or API secrets
